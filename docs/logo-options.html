LOGO DESIGN PROPOSAL

Timothy Yap
个人品牌 Logo 方案

共 10 款方向，每款含设计理由与寓意说明。
配色基准：晨光个人风（#f5a623 · #ff8c69 · #3d2e2a）

选定后告知编号（01–10），将应用于网站 Navbar、Favicon 与简历。

01

Sol 日光

Sun mark + Wordmark

设计理由

圆形太阳 + 射线是最直觉的「阳光、积极」符号，与网站晨光配色一致。字标用 Fraunces 风格衬线，温暖而不 corporate。

寓意

  • 太阳 = 你的 optimistic 个人气质
  • 射线 = 能量向外扩散，像 campaign 触达
  • 适合 Navbar、个人品牌入门方案
友好阳光易识别

02

Arc 上升弧

Growth curve monogram

设计理由

一条从左下到右上的弧线，末端光点 — 像 metrics dashboard 里的 growth line。无太阳，更 professional。

寓意

  • 弧线 = CPL 下降、转化上升
  • 光点 = 目标达成、lead 产生
  • 契合 Performance Marketing 数据叙事
数据感专业

03

Pulse 脉动

Search + heartbeat line

设计理由

放大镜圆（搜索 intent）+ 心电图折线（live data）组合，表达 search-to-conversion 全链路。

寓意

  • 搜索圈 = Google Search 核心工作
  • 脉冲线 = 实时 campaign 监控
  • 蓝 + 金 = 数据 + 人的温度
SearchTracking

04

Monoline T 一笔画

Single-stroke lettermark

设计理由

T 字母一笔完成，末端金色光点。Icon 可单独做 favicon，wordmark 去掉重复 T 形成 Timothy。

寓意

  • 一笔画 = 简洁、直接、不绕弯
  • 光点 = 想法、灵感、optimism
  • 极简主义 personal brand
极简Favicon 友好

05

Pure Wordmark 纯字标

Typography only — no icon

设计理由

没有任何图标，只靠字体、字距和颜色。像 editorial / 杂志标题，最干净。

寓意

  • 名字即品牌 — 自信、成熟
  • 无图形干扰，内容优先
  • 适合求职信、LinkedIn 封面
Editorial干净

06

Funnel Ray 漏斗光

Lead gen funnel symbol

设计理由

漏斗形 + 底部光点 = lead 落入。直接对应你的 lead generation 专长。

寓意

  • 漏斗 = 全漏斗 marketing
  • 底部光 = conversion 完成
  • 行业辨识度最高的一款
Lead GenMarketing

07

Orbit 轨道

Multi-campaign orbit

设计理由

中心圆 + 轨道 + 卫星点 — 象征同时管理 30+ campaigns，多线并行。

寓意

  • 轨道 = 系统化 workflow
  • 卫星 = 多个 active campaigns
  • 动态、busy、高效
Multi-task动态

08

Warm Badge 暖色徽章

Circular personal stamp

设计理由

圆形徽章像 personal stamp / 个人印章，T 居中，YAP 小字在圆内 — 比 TY 方块精致。

寓意

  • 徽章 = 认证感、可信度
  • 圆形 = 完整、和谐、friendly
  • 适合名片、email signature
可信Stamp

09

Dot Matrix 点阵

Data pixel T

设计理由

点阵组成 T 形 — 像 GA4 报表、pixel tracking、数据可视化。

寓意

  • 点 = data points、metrics
  • 矩阵 = 系统化分析
  • 底部橙点 = 转化 highlight
DataAnalytics

10

Wave 挥手

Friendly human line

设计理由

与网站 About 区人物插画同一语言 — 简约线条人形 + 挥手，强调 approachable。

寓意

  • 人形 = 真实的人，不是机器
  • 挥手 = 好相处、愿意沟通
  • 与「阳光个人风」最情感化的一款
HumanFriendly
